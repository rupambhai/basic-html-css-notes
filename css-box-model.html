CSS Box Model – Basic Tutorial

Every HTML element on a web page is a box. The CSS Box Model describes how that box is structured:

+-----------------------------+
|           Margin            |
|  +-----------------------+  |
|  |        Border         |  |
|  |  +-----------------+  |  |
|  |  |     Padding      | |  |
|  |  |  +-----------+   | |  |
|  |  |  |  Content  |   | |  |
|  |  |  +-----------+   | |  |
|  |  +-----------------+  |  |
|  +-----------------------+  |
+-----------------------------+

1. Content

The content area is where your text or images appear.

<div class="box">Hello World</div>

.box {
  width: 200px;
  height: 100px;
  background-color: lightblue;
}

2. Padding

Padding adds space inside the box, between the content and the border.

padding: 20px;

You can control each side individually:

padding-top: 10px;
padding-right: 15px;
padding-bottom: 20px;
padding-left: 25px;

Or use shorthand:

padding: 10px 15px 20px 25px; /* top right bottom left */

3. Border

Border adds a line around the padding and content.

border: 2px solid black;
border-top: 2px dashed red;
border-bottom: 5px dotted green;

4. Width and Height

These control the content size only by default. Padding and border add to the total size unless changed.

.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 2px solid black;
}

Total size calculation:

  • Width: 200 + 20(left) + 20(right) + 2(border left) + 2(border right) = 244px
  • Height: 100 + 20(top) + 20(bottom) + 2(border top) + 2(border bottom) = 144px

5. box-sizing

By default, padding and border are added outside the width/height. You can fix this with:

box-sizing: border-box;

Now the width and height include padding and border. Total size stays 200x100.

Example (All Together)

<div class="box">Box Model</div>

.box {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 5px solid darkblue;
  background-color: lightyellow;
  box-sizing: border-box;
}

Quick Recap

PartDescription
ContentActual text/images inside the box
PaddingSpace inside the box, around content
BorderLine around the padding
Width/HeightSize of content (unless using border-box)
