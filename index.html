üìÑ HTML Introduction

HTML (HyperText Markup Language) is the standard language used to create web pages.

						<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

üè∑Ô∏è Common HTML Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a href=""> - Link
  • <img src=""> - Image
  • <ul>, <ol>, <li> - Lists
  • <div> and <span> - Containers
  • <form>, <input>, <button> - Forms

üé® CSS Introduction

CSS (Cascading Style Sheets) is used to style HTML elements.

						<style>
h1 {
  color: blue;
  font-size: 30px;
}
</style>

You can also use external CSS:

						<link rel="stylesheet" href="style.css">

üéØ CSS Selectors

  • h1 - targets all <h1> tags
  • .class - targets all elements with that class
  • #id - targets a specific element with that ID 
						.box {
  background: yellow;
}

#header {
  font-size: 24px;
}

üì¶ CSS Box Model

Each element is a box made of:

  • Content
  • Padding (inside spacing)
  • Border
  • Margin (outside spacing)
						.card {
  width: 200px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}
